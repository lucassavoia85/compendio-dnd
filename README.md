# Compêndio Interativo de D&D 5ª Edição

Uma aplicação web simples e elegante que serve como um guia de referência rápida para as raças e classes básicas de Dungeons & Dragons 5ª Edição. O projeto foi desenvolvido para demonstrar a manipulação do DOM com JavaScript puro e estilização com CSS moderno.

## 🎯 Funcionalidades

- **Visualização em Cards**: Todas as raças e classes são apresentadas em cards individuais, organizados em seções.
- **Busca Dinâmica**: Filtre os cards em tempo real digitando o nome de uma raça ou classe no campo de busca.
- **Design Temático**: Interface com fontes e ícones que remetem ao universo de D&D.
- **Responsividade**: O layout se adapta a diferentes tamanhos de tela, de desktops a dispositivos móveis.
- **Links Externos**: Cada card contém um link para a respectiva página no D&D Beyond para informações mais detalhadas.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

- **HTML5**: Estrutura semântica da página.
- **CSS3**: Estilização completa, utilizando Flexbox, Grid Layout e Variáveis CSS para um design moderno e responsivo.
- **JavaScript (ES6+)**: Manipulação dinâmica do DOM para gerar e filtrar o conteúdo dos cards sem a necessidade de frameworks.
- **RPG Awesome**: Biblioteca de ícones temáticos para RPG.
- **IA Gemini**: Analise e correção de código.
